## Description

### Describe the bug

move line up and move line down commands cease to work when editing in the kanban board


### Expected Behavior

move line up and move line down should work 

### Steps to reproduce

- begin in markdown mode - the move line up and move line down commands work
- switch to kanban mode. add a new card. 
- switch back to markdown mode - the move line up and move line down commands no longer work. not the hot keys and not from the command palette
- close and reopen the vault or switch to another note and back, and now the move line up and move line down commands work again

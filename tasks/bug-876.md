## Description

### Describe the bug

When checkbox is clicked this triggers strikethrough on text and checkmark on text. When a touch event is triggered on mobile, this can also trigger focus. 

To reproduce create a card with two checkboxes and select them after submitting the card. Sometimes strikethrough align with checkbox sometimes one appears when the other does not
